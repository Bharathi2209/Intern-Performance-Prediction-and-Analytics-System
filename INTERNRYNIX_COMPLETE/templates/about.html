🚀 Sprint Journey
S1

Sprint 1 — Data Foundation

Generated 800 intern records with 21 features. Applied 6-step cleaning pipeline resolving 2048 missing values. Saved raw and cleaned datasets separately. Performed EDA with visualizations.

S2

Sprint 2 — Model Development

Trained Random Forest and Gradient Boosting for both classification and regression. Compared Bagging vs Boosting ensemble strategies. Achieved 93%+ accuracy.

S3

Sprint 3 — Optimization & Evaluation

Added 6 engineered features (academic_index, soft_skill_avg, work_efficiency, reliability_score, experience_score, potential_index). Tuned hyperparameters. Pushed accuracy to 98.75%.

S4

Sprint 4 — Model Deployment

Built Flask web application with single intern prediction form, CSV batch upload, and REST API endpoint. Deployed publicly on Render for live access.
